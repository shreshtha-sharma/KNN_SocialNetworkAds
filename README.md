# 📊 Customer Purchase Prediction using KNN

## 🚀 Overview

This project uses the K-Nearest Neighbors (KNN) algorithm to predict whether a customer will purchase a product based on demographic features such as age, estimated salary, and gender.

---

## 🧩 Problem Statement

Businesses need to identify potential customers who are more likely to purchase a product in order to:

* Improve targeting strategies
* Optimize marketing efforts
* Increase conversion rates

---

## 💡 Approach

* Applied **K-Nearest Neighbors (KNN)** classification algorithm
* Used **Euclidean distance (Minkowski, p=2)** as the similarity metric
* Split dataset into training and testing sets
* Evaluated performance using classification metrics

---

## ⚙️ Tech Stack

* Python
* scikit-learn
* Pandas, NumPy
* Matplotlib, Seaborn

---

## 📊 Model Performance

### 🔹 Training Set

* Accuracy: **91%**
* Precision (avg): **0.90+**
* Recall (avg): **0.91**

### 🔹 Test Set

* Accuracy: **95%**
* Precision (avg): **0.93–0.98**
* Recall (avg): **0.95**

---

## 📈 Evaluation Metrics
